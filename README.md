# 📝 Google Docs Quick Prepender
**〜思いついた瞬間に、ドキュメントの先頭へメモを放り込むツール〜**

このツールは、指定したGoogleドキュメントの**一番上の行**に、日時（JST）付きでテキストを追記するためのGoogle Apps Script（GAS）ウェブアプリケーションです。[cite: 1]
ブラウザからサクッと入力して、ドキュメントを開く手間なく情報を蓄積できます。[cite: 1, 2]

---

## ✨ 主な機能
* **ドキュメント選択の柔軟性**：
    * 起動時は「最近使用した5件」を素早く表示します。[cite: 1, 2]
    * 必要に応じて「すべてのドキュメント」を取得でき、名前順（日本語対応）でソートされます。[cite: 1, 2]
* **スマートな追記（Prepending）**：
    * ドキュメントの末尾ではなく、**先頭（0番目の位置）**に挿入されます。[cite: 1]
    * 最新のメモが常に一番上に来るため、確認が容易です。[cite: 1]
* **自動タイムスタンプ**：
    * 日本時間（JST）で「【yyyy/MM/dd HH:mm】」という形式のヘッダーが自動で付与されます。[cite: 1]
* **快適なUI/UX**：
    * 目に優しいダークモードを採用。[cite: 2]
    * スマートフォンからの入力を考慮したレスポンシブ設計（viewport設定済み）。[cite: 1]
    * 保存中はボタンが無効化され、二重送信やエラーを防止します。[cite: 2]

---

## 🤖 AIツールとの最強連携
このツールでGoogleドキュメントに情報を集約することで、最新のAIを最大限に活用できます。

* **NotebookLM で活用**
    * Google ドキュメントをソースとして登録すれば、ドキュメントを更新した後に NotebookLM 側で**「更新（再取得）」ボタンを1クリック**するだけで、最新のメモに基づいた回答が得られます。
* **Gemini で活用**
    * Google Workspace 拡張機能を利用して、Gemini から直接ドキュメントを読み込めます。アップロードの手間なく、常に最新情報に基づいた分析や要約が可能です。

## 🚀 セットアップ方法

1.  **Google Apps Scriptプロジェクトの作成**
    * [Google Apps Script](https://script.google.com/)で新しいプロジェクトを作成します。
2.  **ファイルの配置**
    * プロジェクト内のスクリプトファイルに **音声入力用スクリプト.gs** の内容を貼り付けます。[cite: 1]
    * 左側の「＋」から「HTML」を選択し、ファイル名を **index.html** （拡張子不要、中身は `index.html` の内容）にして保存します。[cite: 1, 2]
3.  **デプロイ**
    * 「新しいデプロイ」をクリックし、種類を「ウェブアプリ」に設定します。[cite: 1]
    * **設定**：
        * 次のユーザーとして実行：自分
        * アクセスできるユーザー：自分のみ（推奨）
4.  **承認**
    * デプロイ後に発行されたURLにアクセスし、Googleドライブおよびドキュメントへのアクセス権限を承認します。[cite: 1]

---

## 🛠 使い方
1.  ウェブアプリのURLを開きます。
2.  **「📂 追記先のドキュメント」**から対象のファイルを選択します。[cite: 2]
3.  **「📝 入力内容」**にメモしたいテキストを入力します。[cite: 2]
4.  **「📄 文書の先頭に保存」**をクリックすると、即座にドキュメントへ反映されます。[cite: 1, 2]
