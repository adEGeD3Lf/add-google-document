# 📝 Google Docs Quick Prepender
**〜思いついた瞬間に、ドキュメントの先頭へメモを放り込むツール〜**

このツールは、指定したGoogleドキュメントの**一番上の行**に、日時（JST）付きでテキストを追記するためのGoogle Apps Script（GAS）ウェブアプリケーションです。
ブラウザからサクッと入力して、ドキュメントを開く手間なく情報を蓄積できます。

---

## ✨ 主な機能
* **ドキュメント選択の柔軟性**：
    * 起動時は「最近使用した5件」を素早く表示します。
    * 「すべてのドキュメントを表示」ボタンで全件取得ができ、名前順（日本語対応）でソートされます。
* **スマートな追記（Prepending）**：
    * ドキュメントの末尾ではなく、**先頭（0番目の位置）**に挿入されます。
    * 最新のメモが常に一番上に来るため、スクロール不要で確認が容易です。
* **自動タイムスタンプ**：
    * 日本時間（JST）で「【yyyy/MM/dd HH:mm】」という形式のヘッダーが自動で付与されます。
* **快適なUI/UX**：
    * 目に優しいダークモードを採用。
    * スマートフォンからの入力を考慮したレスポンシブ設計（viewport設定済み）。
    * 保存中はボタンが無効化され、二重送信やエラーを防止します。

---

## 🤖 AIツールとの最強連携
このツールでGoogleドキュメントに情報を集約することで、最新のAIを最大限に活用できます。

* **NotebookLM で活用**
    * Google ドキュメントをソースとして登録すれば、ドキュメントを更新した後に NotebookLM 側で**「更新（再取得）」ボタンを1クリック**するだけで、最新のメモに基づいた回答が得られます。
* **Gemini で活用**
    * Google Workspace 拡張機能を利用して、Gemini から直接ドキュメントを読み込めます。アップロードの手間なく、常に最新情報に基づいた分析や要約が可能です。

## 🚀 セットアップ方法

1.  **Google Apps Scriptプロジェクトの作成**
    * [Google Apps Script](https://script.google.com/)で新しいプロジェクトを作成します。
2.  **ファイルの配置**
    * プロジェクト内のスクリプトファイルに **音声入力用スクリプト.gs** の内容を貼り付けます。
    * 左側の「＋」から「HTML」を選択し、ファイル名を **index.html** （拡張子不要、中身は提供されたHTMLコード）にして保存します。
3.  **デプロイ**
    * 「新しいデプロイ」をクリックし、種類を「ウェブアプリ」に設定します。
    * **設定**：
        * 次のユーザーとして実行：自分
        * アクセスできるユーザー：自分のみ（推奨）
4.  **承認**
    * デプロイ後に発行されたURLにアクセスし、Googleドライブおよびドキュメントへのアクセス権限を承認します。

---

## 🛠 使い方
1.  ウェブアプリのURLを開きます。
2.  **「📂 追記先のドキュメント」**から対象のファイルを選択します。
3.  **「📝 入力内容」**にメモしたいテキストを入力します。
4.  **「📄 文書の先頭に保存」**をクリックすると、即座にドキュメントへ反映されます。

---

## 📝 技術仕様
* **言語**: Google Apps Script (JavaScript), HTML5, CSS3
* **使用API**: 
    * `DriveApp`: Googleドキュメント形式のファイル一覧取得。
    * `DocumentApp`: 指定IDのドキュメントオープンおよび段落（0番目）への挿入。
* **タイムゾーン**: `JST` (日本標準時)
* **エラーハンドリング**: ファイル未選択や空入力時のアラート、GAS側でのtry-catch処理を実装済み。
